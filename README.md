# Weather Dashboard

This project was developed as part of Task 3 of the ApexPlanet Web Development Internship.

## Features

- 🌍 Search weather by city
- 🌡 Displays temperature
- 💧 Displays humidity
- 🌬 Displays wind speed
- ☁ Shows weather condition with icons
- 🖼 Interactive image carousel
- 📱 Responsive design using media queries
- ⌨ Press Enter to search
- ❌ Invalid city handling

## Technologies Used

- HTML
- CSS
- JavaScript
- Open-Meteo API
